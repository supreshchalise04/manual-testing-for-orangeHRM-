# OrangeHRM Login Page Manual Test Cases

This repository contains **manual test cases** for the login functionality of the [OrangeHRM demo site](https://opensource-demo.orangehrmlive.com/web/index.php/auth/login). The test cases are documented in an Excel file for easy reference.

## Table of Contents

- [Tools Used](#tools-used)
- [Author](#author)

## Overview
The login page of OrangeHRM allows users to authenticate using a username and password. This repository contains **10 detailed manual test cases** that validate various login scenarios, including positive and negative test cases.

## Test Scope
The testing focuses on:
- Valid login with correct credentials
- Invalid login attempts with wrong credentials
- UI elements validation (username field, password field, login button, error messages)
- Functional behavior of the login process

## Test Cases
The Excel file includes the following columns:
- **Test Case ID**: Unique identifier for each test case
- **Test Scenario**: Description of the feature or functionality being tested
- **Test Steps**: Step-by-step instructions to execute the test
- **Test Data**: Input data used for testing
- **Expected Result**: Expected outcome of the test
- **Actual Result**: Outcome observed during testing


## Tools Used
- **Browser**: Google Chrome / Firefox
- **Excel**: Microsoft Excel for documenting test cases
- **Test Environment**: [OrangeHRM Demo](https://opensource-demo.orangehrmlive.com/web/index.php/auth/login)
